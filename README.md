## 项目介绍
基于springboot的税务管理系统：前端 jsp、jquery，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户、税务人员；集成政策查询、公告、申报税务、报税记录、申请发票等功能于一体的系统。
